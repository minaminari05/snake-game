# Snake Game

A simple Snake game implemented using HTML, CSS, and JavaScript. 

## How to Play
- Use the arrow keys to control the snake.
- Eat the red apples to grow and increase your score.
- Avoid running into the walls or your own body.
- Try to get the highest score!

## Features
- Top 5 ranking system (local to your browser)
- Responsive and simple UI
- Restart button and username input

## GitHub Pages Deployment
You can play this game online by enabling GitHub Pages:

1. Go to your repository settings on GitHub.
2. Scroll down to the "Pages" section.
3. Under "Source", select the `main` branch and set the root (`/`) as the folder.
4. Save. After a few moments, your game will be available at:
   `https://minaminari05.github.io/snake-game/`

Enjoy the game!
